Layout (Teamwork)¶

Using these options you can create a start page for a Teams teamwork. First you see the list of the layouts created so far, for example:

To create a new layout, do the following:

  1. Click the plus.

  2. Add a title and an alias (as usual, click the flag to change language):

  1. When the layout has been created, click the name in the list.

  2. Build the start page using available sections and blocks.

You do this the same way as you work with the layout for Workspace, see the heading “Working with the layout” on this page: Navigation bar
